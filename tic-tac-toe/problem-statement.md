## Tic-Tac-Toe
Build a fully functional 2-player Tic-Tac-Toe game on a 3×3 grid.

### Phase 1 — Game Play
Two players take turns: Player X always goes first
A player clicks an empty cell to place their mark (X or O)
Clicking an already-occupied cell does nothing
Display whose turn it is at all times

### Phase 2 — Win / Draw Detection
After each move, check for a winner
Win condition: 3 matching marks in a row, column, or diagonal
If a winner is found, highlight the winning cells and display a winner banner
If all 9 cells are filled with no winner, display a draw message

### Phase 3 — Reset
A "Restart" button resets the board and starts a new game from scratch
Player X always goes first after a reset
